# Solana Atlas

Interactive 3D knowledge graph of the Solana ecosystem — 1001 terms, 14 categories, built on [`@stbr/solana-glossary`](https://github.com/solanabr/solana-glossary).

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16 (App Router)
- react-force-graph-3d + Three.js
- Zustand
- Tailwind CSS
